## Domains

### Exceptions

-   / -> Home
-   /join -> 회원가입
-   /login -> 로그인
-   /search -> 검색

---

-   /users/:id -> 회원정보 조회
-   /users/logout -> 로그아웃
-   /users/edit -> 회원정보 수정
-   /users/delete -> 회원 탈퇴

---

-   /videos/:id -> 동영상 시청
-   /videos/:id/edit -> 동영상 수정
-   /videos/:id/delete -> 동영상 삭제
-   /videos/upload -> 동영상 게시
